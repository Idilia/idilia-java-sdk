* it was created using m2eclipse plugin using by doing new project->maven->maven project and selecting maven-archetype-quickstart
* it uses mvn 3.0.4, mvn can be called with following targets: compile, test, package, install
* it's a pre-requisite to web_test_suite and ldee webapps, so it has to be installed first by: mvn install

Eclipse:
- install the plugin m2e if not done already (maven eclipse integration)
- convert the project to maven if not yet: project right click -> configure -> convert to maven project
